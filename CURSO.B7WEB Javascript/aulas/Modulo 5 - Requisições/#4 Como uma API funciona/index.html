Como uma API funciona? - Modulo 5

API = Application Programming Interface

Nada mais é do que uma forma de fazer comunicação entre um software e outro software.

Pode ser: Um navegador, um aplicativo de telemovel, um servidor.. Enfim, não importa. Entre um sistema e outro precisa haver uma comunicação.

JSON = Javascript Object Notation
E a forma como se organiza os arquivos para enviar para outra maquina na qual está tentando se comunicar.

Link do {JSON} Placeholder
Feito para testar e prototipar projetos que usarão API futuramente. :)

O que é um Callback - Modulo 5

Callback = Ligar de volta

Callback é quando uma função é criada, e essa função é passada para alguém em uma parte do código, e a essa parte, vai decidir "QUANDO" e "QUE HORA" ela vai executar essa função.

Resumindo: É uma função criada para ser executada no futuro
